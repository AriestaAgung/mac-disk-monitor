# Mac Disk Monitor

This is a simple disk monitor placed on your top menu bar. This app is a learning result from https://www.youtube.com/watch?v=d4qwt3zcLvA&ab_channel=KarinPrater.

# Built on

- Xcode 16.2
- macOS Sequoia 15.3.1
